# java
Java泛型不支持基本类型，即ArrayList<int>这样的代码是不允许的

泛型类
声明类名后使用<E>（E可以是任何其他字母），即可以指定该类是一个泛型类

泛型方法
public <E> E getX(E x){
        return x;
}
1、泛型方法可以在泛型类里面定义，也可以在非泛型类里面定义
2、非泛型类里面定义泛型方法：只需将泛型参数列表置于返回值前。
3、泛型类里面的泛型方法:无需再声明泛型
